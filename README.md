# Image Gallery

A modern, interactive horizontal scrolling image gallery built with HTML, CSS, and JavaScript.

## Live Demo
Try it live here: [Image Gallery](https://asma-ehsan.github.io/Image-Gallery/)

## Features
- Horizontal scrolling with mouse wheel.
- Smooth scrolling with left/right navigation buttons.
- Grayscale images by default, scale and color on hover.
- Responsive layout using CSS Grid.
- Interactive navigation arrows for better control.

## How to Use
1. Open `index.html` in your browser.
2. Scroll horizontally using your mouse wheel or trackpad.
3. Use the left (`<`) and right (`>`) buttons to navigate through images.
4. Hover over images to see the scale and color effect.

## What I Learned
- CSS Grid for flexible image layout.
- Styling hover effects with transitions and transforms.
- Handling horizontal scrolling with JavaScript (`scrollLeft` and `scrollBehavior`).
- Adding interactive UI elements like navigation buttons.
- Combining CSS and JavaScript to create a smooth, user-friendly gallery experience.
